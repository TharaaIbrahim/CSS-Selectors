CSS Selectors

  • Class Selector:

    • It selects element with a specific class attribute.

  • ID Selectror:

    • Selects the value of the ID attribute and use (#) symbol before the value.

  • Element Selector:

    • Use the element as a selector.

  • Universal Selector:

    • Is used for selecting all elements with (*) symbol

  • Descendant Selector:

    • Matches all elements that are descendants of a specified element.

  • Child Element:

    • Selects all elements that are the children of a specified element.
